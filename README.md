# 🧠 BrainSync AI

> **AI-Powered Study Companion** — Upload your notes, get instant summaries, flashcards, quizzes, and chat with your documents using RAG.

## 🚀 Features

- 📄 **Note Upload** — Upload PDFs or text notes
- 🤖 **AI Summarizer** — Instant AI-generated summaries
- 🃏 **Flashcard Generator** — Auto-create Q&A flashcards
- 📝 **Quiz Engine** — Multiple-choice quizzes with scoring
- 💬 **Chat with Notes** — RAG-powered chat interface
- 📊 **Progress Tracker** — Study streaks & performance dashboard

---

## 🛠️ Tech Stack

```
Frontend  →  Flutter (Dart)
Backend   →  FastAPI (Python)
AI        →  OpenAI GPT-4 + ChromaDB (RAG)
Database  →  PostgreSQL
Auth      →  JWT
```
